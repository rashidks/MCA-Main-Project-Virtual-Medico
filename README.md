# MCA Main Project – Virtual Medico

## 🏥 Overview

**Virtual Medico** is an intelligent healthcare management system developed as my **Master of Computer Applications (MCA) Main Project**.

The system provides a centralized platform for hospitals, pharmacies, doctors, and patients. It also integrates a **Machine Learning-based Lung Cancer Prediction** module to assist in early disease prediction.

---

## ✨ Features

- 🏥 Hospital Management
- 👨‍⚕️ Doctor Management
- 📅 Online Doctor Appointment Booking
- 💊 Online Pharmacy Management
- 🔍 Medicine Availability Search
- 📍 Hospital & Pharmacy Information
- 📝 Patient Registration & Login
- 🔔 Appointment Notifications
- 🤖 Machine Learning-Based Lung Cancer Prediction
- 👨‍💼 Admin Dashboard

---

## 👥 User Modules

### Admin
- Manage hospitals
- Verify pharmacy registrations
- Manage complaints and feedback
- System administration

### Hospital
- Manage doctors
- Add hospital facilities
- Manage healthcare packages

### Pharmacy
- Register pharmacy
- Update medicine availability

### Patient
- Register and log in
- Search hospitals and doctors
- Book appointments
- Search medicines
- Access lung cancer prediction
